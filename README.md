FengShui_FightTracker
=====================

Simple app for FengShui table-top rpg to track fight statistics for both named and mooks NPCs
It's made of a single simple activity with a custom listView. 
Buttons allow to add characters (named or mooks) with their own stats to the custom adapter.

The cutsom view for each element allow to make roll or change values such as segment, wounds or number.

Their is a global segment counter and character which are supposed to play are highlighted
